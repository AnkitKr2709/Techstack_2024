/* Select the 10th paragraph inside each div */
div p:nth-of-type(10) {
    background-color: green; /* Example background color */
    color: white;            /* Example text color */
    font-weight: bold;       /* Example additional styling */
}
Explanation
The selector div p:nth-of-type(10) targets the 10th <p> element inside each <div>.
You can replace background-color, color, and font-weight with any other CSS properties and values you want to apply to the 10th paragraph.
